# battleship
A simple web implementation of a classic board game.

## Development Setup
1. Install [Node.js](https://nodejs.org/en/);
2. Execute:
	```
	npm install
	npm install -g gulp-cli bower
	gulp init
	```
3. To Build the project run:
	```
	gulp build
	```
4. To start server:
	```
	npm start
	```
